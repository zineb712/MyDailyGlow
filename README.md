## Description
Application web interactive pour gérer ses routines beauté, avec suivi personnalisé et interface simple.  
Back-end en Flask et base de données SQLite, front-end en HTML/CSS.

## Technologies utilisées
- Python, Flask
- SQLite
- HTML, CSS
- PythonAnywhere (déploiement)

## Fonctionnalités
- Création et suivi des routines beauté
- Interface responsive
- Gestion des données utilisateurs dans SQLite
